Wave Manager Tool for controlling the Gameplay Loop & flow of a Tower Defense game.

Functionalities:

The user creates a Wave Config. Our game has a set number of waves per level(5). The user specifies what enemies should be released per pulse and how many pulses there are per wave. A pulse is which and how many enemies to be released simultaniously. A wave is completed when all enemies within that wave are defeated.

Tweakable Values and buttons:

* Save created configs to binary

* Load created configs from binary

* Start Waves

* Restart Waves

* Preparation Time(Time before first pulse in a wave)

* Time To Next Pulse

* Number of Enemies per pulse

* Max amount of enemies per Wave (Saved in EnemyPool)
